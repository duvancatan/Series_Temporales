\section{Introducción}
\medskip
Las series de tiempo económicas y financeiras presentan ciertas características especiales. Una de ellas es el llamado agrupamiento de volatilidades, significando que la estructura de la varianza en la serie cambia con el tiempo y depende de la información pasada. Por tanto, diferentes modelos econométricos son utilizados para estimar y cuantificar las posibles ganancias o pérdidas que se pueden producir en los mercados financieros.

\bigskip

\section{Objetivo del Curso}
\medskip
Presentar métodos para el análisis de series de tiempo principalmente en las áreas de economía y finanzas, tanto desde una perspectiva teórica como aplicada. Al terminar este curso, los estudiantes tendrán la capacidad de entender y tratar con datos de series de tiempo financieras y ser capaces de estimar, interpretar e implementar los modelos econométricos univariados y multivariados, con el apoyo del software estadístico $\mathrm{R}-$project.

\section{Contenido del curso}
\medskip
\begin{enumerate}

\item[1.] \textbf{Procesos Estacionarios.} 
\begin{itemize}
\item Funciones de Autocovarianza y Autocorrelación $(ACF).$
\item Función de Autocorrelación Parcial $(PACF).$
\item Modelos $ARIMA.$
\item Identificación y Pronóstico.
\end{itemize}

\item[2.] \textbf{Teoría Espectral en Series de Tiempo.}
\begin{itemize}
\item Densidad Espectral.
\item Periodograma.
\item Pruebas para Componentes Periódicos.
\item Espectro Suavizado.
\item Filtros Lineales.
\end{itemize}

\item[4.] \textbf{Modelos con Memoria Larga.}
\begin{itemize}
\item Introducción.
\item Pruebas para Detección de Memoria Larga.
\item Modelos $ARFIMA$.
\item Identificación y Pronóstico.
\end{itemize}

\item[3.] \textbf{Modelos para la Volatilidad.}
\begin{itemize}
\item Datos Financieros.
\item Varianza Condicional.
\item Modelos $ARCH$ y $GARCH$.
\item Identificación y Pronóstico.
\item Modelos de Volatilidad Estocástica.
\end{itemize}

\item[5.] \textbf{Métodos Lineales Multivariados.}
\begin{itemize}
\item Introducción. 
\item Estacionaridad.
\item Modelo Autorregresivo Vectorial $VAR$.
\item Identificación y Estimación.
\item Causalidad de Granger.
\item Pronóstico.
\end{itemize}
\newpage
\item[6.] \textbf{Procesos Cointegrados.}
\begin{itemize}
\item Introducción.
\item Tendencias Comunes.
\item Pruebas de Cointegración.
\item Modelos de Corrección de Errores.
\end{itemize}

\item[7.] \textbf{Modelos GARCH Multivariados.}
\begin{itemize}
\item Introducción.
\item Generalizaciones del Modelo $GARCH$ Univariado.
\item Modelos $EWMA$, $VEC$ y $BEKK.$
\item Modelos Factoriales y Ortogonales.
\end{itemize}

\end{enumerate}
